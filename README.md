# Blog Search React App

A React blog search application with debounced search, highlighted results, and links to CoderByte social platforms.

## Features
- Debounced search for posts
- Highlight search terms in titles and bodies
- Clear search input button
- Links to CoderByte X (Twitter) and LinkedIn pages

## Requirements
- Node.js (v16 or higher recommended)
- npm or yarn

## Installation

1. Clone the repository:


  git clone https://github.com/yourusername/your-repo-name.git
  cd your-repo-name
## Install dependencies:
